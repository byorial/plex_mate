### Plex Mate

** 이 플러그인은 Plex 서버가 실행되는 파일시스템에 접근 가능해야 한다. **

#### 주요기능

  * 썸네일 삭제 및 이미지 파일 저장없이 사용
  * 캐시 자동 삭제
  * 라이브러리별 스캔 스케쥴링
  * DB 제어
  * 특이 메타 검색
  * 스캔
